# Password Generator (CLI) 🔐

A simple but practical **command-line password generator** built in Python, designed to help you learn from the basics to more advanced concepts while using Git and GitHub.

## Features ✅

- Generate passwords of any length.
- Choose password strength:
  - Easy: only lowercase letters.
  - Medium: lowercase + uppercase + numbers.
  - Strong: lowercase + uppercase + numbers + symbols (`!@#$%^&*`).
- Add a **label** to each password (e.g. `Gmail`, `Netflix`, `School account`).
- **Save all passwords** to `passwords.txt` with:
  - Label
  - Strength level
  - Length
  - Password value
- When you run the program, it **shows all saved passwords** before letting you generate a new one.
